## Overview
The system captures patient voice input, processes it using AI models, and stores structured data for healthcare professionals.

## Architecture
- Mobile App (Patient / Health Worker)
- Speech-to-Text Engine
- NLP Processing Layer
- Backend API
- Secure Database
- Doctor Dashboard

## Flow
1. User records symptoms via voice
2. Speech converted to text
3. AI extracts medical information
4. Data stored securely
5. Doctor reviews and responds

## Security
- Encrypted data storage
- Secure authentication
- Role-based access
